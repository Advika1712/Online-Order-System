# Online Order Processing & Tracking System (OOP-TS)

## Overview
The Online Order Processing & Tracking System (OOP-TS) is a Python-based application developed using Object Oriented Programming (OOP) concepts. The system simulates an online shopping platform where users can browse products, add items to a cart, place orders, make payments, and track delivery status.

The project demonstrates modular programming, file handling, data analysis, and visualization techniques using Python libraries.

---

## Features
- User Login Authentication
- Product Display System
- Add to Cart Functionality
- Order Placement & Payment
- Order Tracking
- Sales Report Generation

---

## Technologies Used
- Object Oriented Programming (OOP)
- NumPy
- Pandas
- Matplotlib
- JSON File Handling

## Project Structure

- `user_auth.py` → User login authentication

- `order_models.py` → OOP class definitions

- `order_services.py` → Cart, order, payment & tracking operations

- `data_manager.py` → JSON file handling, reports & data analysis

- `app.py` → Main menu-driven interface

- `__init__.py` → Package initialization file
